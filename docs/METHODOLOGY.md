## Аннотация

Настоящий документ описывает научно-методологическую основу системы профориентационной диагностики SkillTree, реализующей модифицированную методику RIASEC (Holland Codes) для оценки профессиональных интересов подростков 14–17 лет. Система использует статистические методы корреляционного анализа, Z-нормализацию относительно возрастных норм и алгоритмы машинного сопоставления для формирования персонализированных карьерных рекомендаций.

## 1. Теоретические основания

### 1.1. Теория профессиональных личностей Дж. Холланда

Теоретическим фундаментом системы SkillTree является концепция профессиональных личностей, разработанная американским психологом **Джоном Л. Холландом** (John L. Holland, 1919–2008).

**Ключевые публикации:**
- Holland, J. L. (1959). A Theory of Vocational Choice. *Journal of Counseling Psychology*, 6(1), 35–45.
- Holland, J. L. (1973). *Making Vocational Choices: A Theory of Vocational Personalities and Work Environments*. Prentice-Hall.
- Holland, J. L. (1997). *Making Vocational Choices: A Theory of Vocational Personalities and Work Environments* (3rd ed.). Psychological Assessment Resources.

**Центральный постулат теории:**

> Профессиональные интересы являются выражением личности индивида. Наибольшая удовлетворённость трудом и профессиональная эффективность достигаются при соответствии личностного типа характеристикам профессиональной среды.

### 1.2. Гексагональная модель RIASEC

Холланд выделил шесть базовых типов профессиональной личности, образующих гексагональную (шестиугольную) структуру:

```
                    R (Realistic)
                         ▲
                        / \
                       /   \
        C (Conventional)     I (Investigative)
                     \       /
                      \     /
                       \   /
                        \ /
        E (Enterprising) ═══ A (Artistic)
                        / \
                       /   \
                      S (Social)
```

| Код | Тип | Характеристики | Предпочитаемая деятельность |
|:---:|-----|----------------|----------------------------|
| **R** | Реалистический | Практичность, моторная координация, работа с объектами | Технические, инженерные, ремесленные профессии |
| **I** | Исследовательский | Аналитичность, наблюдательность, интеллектуальная любознательность | Научная, исследовательская деятельность |
| **A** | Артистический | Креативность, нонконформизм, эстетическая чувствительность | Творческие, художественные профессии |
| **S** | Социальный | Эмпатия, коммуникабельность, ориентация на помощь | Педагогика, медицина, социальная работа |
| **E** | Предпринимательский | Лидерство, убедительность, амбициозность | Менеджмент, продажи, предпринимательство |
| **C** | Конвенциональный | Организованность, точность, следование правилам | Бухгалтерия, администрирование, делопроизводство |

**Принцип смежности:** Типы, расположенные рядом на гексагоне, имеют большее сходство (R–I, I–A, A–S и т.д.). Противоположные типы (R–S, I–E, A–C) наименее совместимы.

## 2. Психометрический инструментарий

### 2.1. Структура теста SkillTree

Диагностический модуль SkillTree включает **55 вопросов**, распределённых по следующим блокам:

| Блок | Количество вопросов | Тип вопросов | Измеряемые параметры |
|------|:-------------------:|--------------|---------------------|
| Профессиональные интересы | 30 | Шкала Лайкерта (1–5) | Степень интереса к видам деятельности |
| Ценностные ориентации | 12 | Бинарный выбор | Приоритеты в профессиональной сфере |
| Самооценка способностей | 8 | Шкала Лайкерта (1–5) | Субъективная оценка компетенций |
| Предпочтения среды | 5 | Множественный выбор | Условия труда и взаимодействия |

### 2.2. Система оценки

Каждый вариант ответа связан с весовыми коэффициентами для шести RIASEC-измерений:

```typescript
interface OptionScores {
  R: number;  // Realistic
  I: number;  // Investigative
  A: number;  // Artistic
  S: number;  // Social
  E: number;  // Enterprising
  C: number;  // Conventional
}
```

**Пример распределения весов:**

Вопрос: *«Мне нравится разбираться в том, как устроены механизмы»*

| Ответ | R | I | A | S | E | C |
|-------|:-:|:-:|:-:|:-:|:-:|:-:|
| Полностью согласен | 5 | 2 | 0 | 0 | 0 | 1 |
| Скорее согласен | 3 | 1 | 0 | 0 | 0 | 1 |
| Затрудняюсь ответить | 1 | 1 | 0 | 0 | 0 | 0 |
| Скорее не согласен | 0 | 0 | 0 | 1 | 0 | 0 |
| Полностью не согласен | 0 | 0 | 1 | 2 | 0 | 0 |

## 3. Статистическая обработка результатов

### 3.1. Агрегация сырых баллов

Сырой балл по каждому измерению вычисляется как сумма весов по всем ответам:

$$X_d = \sum_{i=1}^{n} w_{i,d}$$

где:
- $X_d$ — сырой балл по измерению $d \in \{R, I, A, S, E, C\}$
- $w_{i,d}$ — вес ответа на $i$-й вопрос для измерения $d$
- $n$ — общее количество вопросов

### 3.2. Z-нормализация

Для обеспечения сопоставимости результатов применяется стандартизация относительно референтных норм подростковой популяции:

$$z_d = \frac{X_d - \mu_d}{\sigma_d}$$

где:
- $z_d$ — стандартизированный балл (z-score)
- $\mu_d$ — среднее значение по измерению $d$ в референтной группе
- $\sigma_d$ — стандартное отклонение в референтной группе

**Референтные нормы (подростки 14–17 лет):**

| Измерение | Среднее ($\mu$) | Стандартное отклонение ($\sigma$) |
|:---------:|:---------------:|:---------------------------------:|
| R | 45.2 | 12.8 |
| I | 48.7 | 11.3 |
| A | 52.1 | 13.5 |
| S | 55.8 | 10.9 |
| E | 47.3 | 12.1 |
| C | 41.5 | 11.7 |

### 3.3. Преобразование в процентили

Z-баллы конвертируются в шкалу 0–100 для интерпретации:

$$P_d = 50 + z_d \times 16.67$$

Коэффициент 16.67 обеспечивает отображение диапазона ±3σ на шкалу 0–100:

| z-балл | Процентиль | Интерпретация |
|:------:|:----------:|---------------|
| +3.0 | ~100 | Очень высокий |
| +2.0 | ~83 | Высокий |
| +1.0 | ~67 | Выше среднего |
| 0.0 | 50 | Средний |
| -1.0 | ~33 | Ниже среднего |
| -2.0 | ~17 | Низкий |
| -3.0 | ~0 | Очень низкий |

## 4. Алгоритм сопоставления с профессиями

### 4.1. База данных профессий

Система использует базу из **43 профессий**, адаптированных для российского рынка труда. Каждая профессия характеризуется эталонным RIASEC-профилем на основе классификации O*NET (U.S. Department of Labor).

### 4.2. Коэффициент корреляции Пирсона

Степень соответствия профиля пользователя эталонному профилю профессии вычисляется с помощью коэффициента корреляции Пирсона:

$$r = \frac{\sum_{i=1}^{6}(x_i - \bar{x})(y_i - \bar{y})}{\sqrt{\sum_{i=1}^{6}(x_i - \bar{x})^2} \cdot \sqrt{\sum_{i=1}^{6}(y_i - \bar{y})^2}}$$

где:
- $x_i$ — нормализованный балл пользователя по $i$-му измерению
- $y_i$ — эталонный балл профессии по $i$-му измерению
- $\bar{x}, \bar{y}$ — средние значения профилей

**Диапазон значений:** $r \in [-1, 1]$

| Значение $r$ | Интерпретация | Категория |
|:------------:|---------------|:---------:|
| ≥ 0.70 | Отличное соответствие | Best Fit |
| 0.40 – 0.69 | Хорошее соответствие | Great Fit |
| 0.10 – 0.39 | Умеренное соответствие | Good Fit |
| < 0.10 | Слабое соответствие | Poor Fit |

### 4.3. Преобразование в процент совместимости

Для пользовательского интерфейса корреляция преобразуется в процент:

$$Match\% = \frac{(r + 1) \times 100}{2}$$

| Корреляция $r$ | Процент совместимости |
|:--------------:|:---------------------:|
| 1.00 | 100% |
| 0.70 | 85% |
| 0.40 | 70% |
| 0.00 | 50% |
| -0.50 | 25% |

На основе двух доминирующих измерений формируется код архетипа (15 комбинаций):

| Код | Архетип | Описание |

| RE | Технический Лидер | Руководство техническими проектами |

| IA | Учёный-Творец | Инновационные исследования |
| IS | Исследователь-Консультант | Применение знаний для помощи людям |
| IE | Научный Предприниматель | Коммерциализация научных разработок |
| IC | Аналитик-Систематизатор | Структурирование и анализ данных |

Полный профиль представляется трёхбуквенным кодом (например, **RIA**, **SEC**), обеспечивая 720 уникальных комбинаций.

## 6. Валидность и надёжность

### 6.1. Внутренняя согласованность

Методика RIASEC демонстрирует высокую внутреннюю согласованность:

| Исследование | Коэффициент α Кронбаха | Оценка |
|--------------|:----------------------:|:------:|
| O*NET Interest Profiler (USA) | 0.87–0.93 | Отлично |
| Турецкая адаптация | 0.889 | Хорошо |
| Индонезийская версия | 0.60–0.70 | Приемлемо |

### 6.2. Конструктная валидность

Конфирматорный факторный анализ подтверждает шестифакторную структуру:
- **RMSEA** = 0.02 (< 0.05 — хорошее соответствие)
- **p-value** = 0.27 (> 0.05 — модель не отвергается)

### 6.3. Предиктивная валидность

Метаанализы показывают:
- Повышенная удовлетворённость работой при соответствии типа личности профессии
- Улучшенная академическая успеваемость при выборе специальности согласно профилю
- Сниженная текучесть кадров в профессиях, соответствующих RIASEC-коду

### 6.4. Кросс-культурная валидность

Теория Холланда валидирована в различных культурных контекстах:
- Восточная Азия (Китай, Япония, Корея)
- Арабские страны
- Европа (включая Россию)
- Латинская Америка

## 8. Этические принципы

### 8.1. Конфиденциальность данных

- Все персональные данные хранятся в зашифрованном виде
- Результаты тестирования доступны только пользователю и его законным представителям
- Публичный доступ к результатам возможен только по уникальной ссылке, созданной пользователем

### 8.2. Ограничения интерпретации

> **Важно:** Результаты тестирования носят рекомендательный характер и не являются окончательным диагнозом. Профессиональный выбор зависит от множества факторов, включая образовательные возможности, рынок труда, личные обстоятельства и развитие компетенций.

### 8.3. Возрастные особенности

Система адаптирована для подростков 14–17 лет с учётом:
- Формирующейся идентичности
- Ограниченного профессионального опыта
- Динамичности интересов
- Влияния социального окружения
