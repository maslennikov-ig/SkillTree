# Инструкция по настройке SendPulse для SkillTree Bot

## Что это и зачем нужно

SendPulse — это сервис для отправки email. Мы используем его для отправки PDF-отчётов с результатами тестирования родителям учеников.

**Бесплатный тариф**: 12 000 писем в месяц (более чем достаточно для старта).

---

## Пошаговая инструкция

### Шаг 1: Регистрация

1. Перейдите на сайт: https://sendpulse.com/ru
2. Нажмите "Регистрация" (правый верхний угол)
3. Заполните форму:
   - Email (рабочий email, на него придут данные для API)
   - Пароль
   - Имя компании: "SkillTree" или название вашей организации
4. Подтвердите email (придёт письмо со ссылкой)

### Шаг 2: Верификация домена отправителя

1. После входа перейдите в раздел **Настройки** → **Адреса отправителей**
2. Нажмите "Добавить адрес"
3. Введите email для отправки писем (например: `noreply@skilltree.ru` или ваш домен)
4. Подтвердите владение адресом (придёт письмо)

> **Важно**: Без верификации письма могут попадать в спам.

### Шаг 3: Получение API ключей

1. Перейдите в раздел: **Настройки** → **API**
   - Прямая ссылка: https://login.sendpulse.com/settings/#api
2. Вы увидите два значения:
   - **API User ID** (выглядит как: `a1b2c3d4e5f6...`)
   - **API Secret** (выглядит как: `x9y8z7w6v5u4...`)
3. Скопируйте оба значения

### Шаг 4: Отправьте нам данные

Отправьте следующие данные (можно в личные сообщения или защищённым каналом):

```
SENDPULSE_API_USER_ID=<ваш API User ID>
SENDPULSE_API_SECRET=<ваш API Secret>
EMAIL_FROM=<email для отправки, например noreply@skilltree.ru>
EMAIL_FROM_NAME=SkillTree
```

---

## Что мы сделаем с этими данными

1. Добавим их в конфигурацию сервера
2. Протестируем отправку тестового письма
3. Подтвердим, что всё работает

---

## Безопасность

- API ключи хранятся только на защищённом сервере
- Они не попадают в публичный код (Git)
- SendPulse использует OAuth 2.0 для авторизации
